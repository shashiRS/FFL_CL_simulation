6. cm_eval_erg_parser¶

6.1. cErgFile¶

Hint

As this class is used by a developer for parsing a Carmaker simulation result file (erg), see the example repository linked in the “Getting Started” section for more details.

class pycmEval.cm_eval_erg_parser.cErgFile(ergFilePath, cacheWholeFile=True)¶

Class to read erg files (Carmaker result files).

Parameters:

  • ergFilePath – Absolute path to the erg file

  • cacheWholeFile – If true, reads the whole erg file once. If false, it reads each quantity if it is necessary. This may lead to worse performance.

getArrayOfAvailableValues()¶

Get array of available signals. :return: List of strings with all available values by name

getCurrentRecordId()¶

Get the current Package id (floor the current byte-position)

Returns:

Current package ID

getUnitOfValue(valueStr)¶

Get the unit of a value.

Returns:

String representation of the value’s unit. If no value found in erg file with the name valueStr, an empty string is returned!

jumpToRecordById(recordId)¶

Set the current reading position of the erg data file to the package with the id recordId. Raise an exception if the recordID exceeds the total record count.

Returns:

True if succeeded.

readAllSignalValues(signalName)¶

Read all signal values for a given signal.

Parameters:

signalName – Name of the signal

Returns:

List of all values for the given signal

readNextTimeStep()¶

Read the next dataPackage in the erg-file. If the end of the erg-file is reached, the function returns an empty string.

Returns:

values as a dict={‘ValueName’:’Value’,…} for this time step.

readRecord(recordId)¶

recordId – Record ID

The record

Read a single signal value.

Parameters:

  • signalName – Name of the signal

  • recordId – Record ID

Returns:

Value of the signal for given record ID

readWholeErgFile()¶

Read the whole erg file.
